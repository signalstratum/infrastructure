# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it responsibly.

**Email:** security@signalstratum.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Initial Assessment:** Within 7 days
- **Resolution Target:** Within 30 days for critical issues

## Scope

This policy applies to:
- Terraform configurations in this repository
- GitHub Actions workflows
- Any infrastructure managed by this code

## Out of Scope

- Third-party services (Cloudflare, GitHub, 1Password)
- Social engineering attacks
- Denial of service attacks

## Recognition

We appreciate responsible disclosure and will acknowledge contributors in release notes when fixes are published (unless you prefer to remain anonymous).
